# Harvey Houlahan

**Spatial · Simulation · ML** — geospatial ML, GPU simulation and neural surrogates for climate, carbon & nature tech.

- **Day job** — ML engineer at ArborMeta: canopy-height & above-ground-carbon models from paired LiDAR captures, live fire-spread mapping, and the geospatial pipelines behind Australian carbon-credit policy advice (FastAPI · PostGIS · PMTiles · MapLibre).
- **Off the clock** — the same discipline, in a browser tab: real solvers in raw WebGPU, and neural operators trained to emulate them. Physics is the teacher, the network is the student.

## Live demos — [hjhportfolio.com/playground](https://hjhportfolio.com/playground)

| | |
|---|---|
| **[Catchment](https://hjhportfolio.com/catchment)** | A living catchment: shallow-water hydrology, erosion and wind-driven fire — plus a convolutional neural operator trained against the solver itself, running as hand-written WGSL compute passes with a live physics-vs-neural error field. No servers, no runtime deps. |
